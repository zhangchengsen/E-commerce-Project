## Project setup 项目初始化
在根目录下进行
```
npm install
```
接着在vue_api_server页面下也进行
`npm install`
本地需安装mysql数据库 并且导入数据文件

`vue_api_server\db\mydb.sql`
`nodemon app.js`启动项目  
最好本地使用vue ui启动项目
